About This Platform

Research platform for analyzing AI-generated applications across multiple models.

{% endblock %} {% block main_content %}
Purpose & Goals

This platform orchestrates generation and multi-dimensional analysis (security, performance, static/dynamic scanning, AI code review) of thousands of small AI-generated web apps produced by diverse foundation models (Anthropic Claude, OpenAI GPT, Google Gemini, etc.).

  • Benchmark cross-model code generation characteristics
  • Surface systemic security & quality patterns
  • Correlate model capabilities with runtime/container behavior
  • Enable reproducible comparative research at scale
Architecture Highlights
  • Flask + HTMX adaptive UI shell
  • Containerized analyzer microservices (security, performance, AI)
  • Flexible JSON-centric persistence for heterogeneous analysis outputs
  • WebSocket + Celery task pipeline bridging UI & analyzers
  • Aggregated metrics, trends, batch orchestration
Key Components
  • ServiceLocator mediated business orchestration
  • Dynamic port & app allocation (precomputed)
  • Unified task manager abstraction
  • Composable partial-driven page wrappers

See docs/DEVELOPMENT.md for deeper details.

Research Themes
  • Multi-model comparative generation economics
  • Security posture drift across iterations
  • Static vs dynamic tool concordance
  • LLM self-analysis reliability
