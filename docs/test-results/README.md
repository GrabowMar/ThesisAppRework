# Test Results Archive

This directory contains historical test results and validation reports from development cycles.

## Contents

### Template Analysis
- **COMPACT_TEMPLATE_ANALYSIS.md** - Codex-mini compact template test results

### Generation System Testing
- **GENERATION_VERIFICATION_RESULTS.md** - Generation system verification results
- **THREE_APP_TEST_RESULTS.md** - Fresh app generation validation (67% success rate)
- **VALIDATION_RESULTS.md** - General validation results
- **TESTING_NEW_GENERATION.md** - Guide for testing new generation system

### Container & Integration Testing
- **LIVE_CONTAINER_TEST_RESULTS.md** - Docker container testing (3 apps)

### Issue Documentation
- **CRITICAL_ISSUES_FOUND.md** - Frontend-backend endpoint mismatch documentation

## Purpose

These documents serve as:
- Historical records of system behavior and performance
- Baselines for regression testing
- Documentation of known issues and their resolutions
- Reference material for understanding past system states

## Note

For current, active test results, run the test suite in `tests/` using pytest. These archived results document specific historical testing sessions and should not be considered current system state.
